Installation
============

Prerequisites
-------------

lostml requires Python 3.7+ and NumPy.

Installation
------------

Clone the repository:

.. code-block:: bash

   git clone https://github.com/yourusername/lostml.git
   cd lostml

Install the package:

.. code-block:: bash

   pip install -e .

Or install dependencies manually:

.. code-block:: bash

   pip install numpy

Development Installation
------------------------

For development, you may want to install in editable mode:

.. code-block:: bash

   pip install -e .

This allows you to make changes to the code and have them immediately available without reinstalling.
